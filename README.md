# mirt
Simple recreation of Vim
## Preqequisites
- Able to compile with the C++23 standard
- make
## Quickstart
```
make
./mirt [filename]
```

## Usage
- Create a `.mirtrc` file in the same directory as the `mirt` executable
- Use vim motions to navigate the text editor
